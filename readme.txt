This widget is used to get the zodiac sign infromation.

== Description ==

This widget help user to get basic information about his/her star.
- By selecting month and date, they can see their star
- Short description of star is also available.

== Installation ==

1. Upload `zodiac_information_widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets in wordpress admin panel and add widget to desired area.
4. Thats all :)
